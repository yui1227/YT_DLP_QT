# YT_DLP_QT README

## 環境建置

1. **安裝 Python**
   - 請確保系統已安裝 Python 3.12 或以上版本。
   - 可從 [Python 官方網站](https://www.python.org/downloads/) 下載並安裝。

2. **建立虛擬環境 (可選，但建議使用)**
   - 在專案目錄中執行以下指令來建立虛擬環境：
     ```bash
     python -m venv venv
     ```
   - 啟動虛擬環境：
     - Windows:
       ```bash
       .\venv\Scripts\activate
       ```
     - macOS/Linux:
       ```bash
       source venv/bin/activate
       ```

3. **安裝所需套件**
   - 使用 `pip` 安裝專案所需的套件：
     ```bash
     pip install -r requirements.txt
     ```

## 執行專案

1. **執行主程式**
   - 在終端機中執行以下指令來啟動程式：
     ```bash
     python main.py
     ```
